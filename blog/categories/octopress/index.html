部署到Github Pages

生成博文并且部署

rake generate
rake deploy

注： 指令执行后将会生成博文并复制文件到 _deploy/ 目录，然后添加到git， commit、push到master分支。

提交source分支

git add .
git commit -m 'message'
git push origin source
